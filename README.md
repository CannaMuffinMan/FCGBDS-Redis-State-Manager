FCG Bot Defense Kit
A scalable, Redis-backed bot defense system built to fight viewbotting, fake engagement, and automated abuse across streaming platforms.
Core Features:

Real-time hit tracking for IPs, devices, payloads, and token anomalies
Redis-backed state so multiple servers share threat data
Graceful fallback to local memory if Redis is down
Designed for horizontal scaling

This is the same bot defense system used to protect Forever Couch Gang platforms.
The code is now open source. Use it. Improve it. Help kill the bot problem.
